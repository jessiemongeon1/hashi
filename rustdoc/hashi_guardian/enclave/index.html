Module enclave 

Source
Expand description

Core enclave types: Enclave holds all guardian state (immutable config set during operator/provisioner-init, mutable runtime state, and the one-time init scratchpad). Lives in the library so external crates (integration test harnesses, ops tooling) can construct and drive an enclave without going through main.
